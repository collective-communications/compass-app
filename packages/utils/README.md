# @compass/utils

Shared utility functions used across the Culture Compass platform.

## Public API

### Functions

- `requireEnv` — Read a required environment variable or throw `MissingEnvError`
- `optionalEnv` — Read an optional environment variable with a fallback default

### Classes

- `MissingEnvError` — Error thrown when a required environment variable is missing

## Key Dependencies

- None (pure TypeScript utilities)
