# @compass/compass

SVG compass visualization component that renders the four-dimension Culture Compass radar chart.

## Public API

### Components

- `Compass` — React component that renders the interactive compass visualization

### Functions

- `renderCompassToString` — Server-side render the compass to an SVG string (for PDF/report generation)

### Types

- `CompassProps` — Props for the Compass component
- `DimensionScore` — Score data for a single dimension
- `DimensionCode` — Union type of the four dimension codes

## Key Dependencies

- `react` — Component rendering
